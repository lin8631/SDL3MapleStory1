## 项目概述

这是一个使用 SDL3 开发 MapleStory风格游戏的客户端项目。

## 项目结构

```
SDL3MapleStory2/
├── Game/                      # 游戏主代码
│   ├── Components/           # 游戏组件
│   │   ├── Character.cpp     # 角色组件
│   │   ├── Skill.cpp         # 技能组件
│   │   ├── Sound.cpp         # 声音组件
│   │   └── ...
│   ├── Core/                 # 核心系统
│   │   ├── Window.cpp        # 窗口管理
│   │   └── Keyboard.cpp      # 键盘输入
│   ├── PlayerSkill/          # 玩家技能实现
│   │   ├── Skill_*.cpp       # 各技能实现文件
│   │   └── PlayerSkill.cpp   # 技能调用入口
│   ├── UI/                   # 用户界面
│   │   ├── Button.cpp        # 按钮组件
│   │   ├── StatusBar.cpp     # 状态栏
│   │   └── UIBuff.cpp        # BUFF显示
│   ├── Systems/              # 游戏系统
│   │   ├── Render.cpp        # 渲染系统
│   │   └── ...
│   ├── Resources/             # 资源管理
│   │   └── Wz.cpp           # WZ文件加载
│   └── main.cpp              # 程序入口
├── Engine/                   # 第三方引擎
│   ├── SDL/                  # SDL3库
│   └── imgui/               # ImGui界面库
├── Config/                   # 配置文件
│   └── keyboard.txt         # 键盘配置
└── Data/                    # 游戏数据资源
    └── *.wz                  # WZ资源文件
```

## 技术栈

- **图形**: SDL3 Renderer
- **界面**: ImGui
- **音频**: FFmpeg + SDL3 Audio
- **资源格式**: WZ文件格式（冒险岛资源格式）
